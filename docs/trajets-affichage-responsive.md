# Repères UM 2.0 — Trajets, affichage replié et confort visuel

## Affichage replié

La saisie d'une nouvelle situation est repliée par défaut afin de laisser apparaître immédiatement la charge équipe.

La charge équipe actuelle / préparatoire / projetée est également repliée par professionnelle :
- le résumé est visible ;
- les détails ne s'affichent qu'à la demande ;
- le comportement est identique sur PC et téléphone.

## Trajets

Repères UM 2.0 ne stocke pas d'adresse.

Le module trajet permet uniquement de renseigner :
- kilomètres aller estimés depuis l'hôpital ;
- minutes aller estimées ;
- période de trajet ;
- circulation estimée.

Ces informations alimentent la lecture cadre de la charge sans contenir de lieu nominatif.

## Palette

La palette visuelle est orientée confort :
- fond sable / écru ;
- accent sauge ;
- ardoise doux pour le texte ;
- ocre doux pour les vigilances ;
- pas de bleu dominant ;
- pas de bordeaux.
